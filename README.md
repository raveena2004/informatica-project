# Informatica-Project

This Informatica ETL project focuses on extracting data from Oracle source tables, transforming the data using various Informatica transformations, and loading the processed data into target tables for analysis and reporting. In this project, transformations such as Filter Transformation were used to remove unwanted records, Router Transformation was implemented for validation and separating valid and invalid data, Aggregator Transformation was used to perform calculations and summarize data, Rank Transformation helped identify top records based on specific conditions, and Sequence Generator was used to create unique IDs for target records. The project demonstrates core ETL concepts including data extraction, cleansing, validation, transformation, and loading, while improving data quality and enabling efficient business reporting and analytics.
